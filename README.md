# FigWorkAI CLI Releases

本仓库用于发布 FigWorkAI CLI (`fig`) 的预编译二进制文件。

## 安装

```bash
npm install -g figworkai
```

## 下载

从 [Releases](https://github.com/FigWorkAI/fig-release/releases) 页面下载对应平台的二进制文件：

| 平台 | 架构 | 文件 |
|------|------|------|
| Linux | x64 | `fig-linux-x64` |
| Linux | arm64 | `fig-linux-arm64` |
| macOS | x64 | `fig-macos-x64` |
| macOS | arm64 | `fig-macos-arm64` |
| Windows | x64 | `fig-windows-x64.exe` |

## 源码

主仓库：https://github.com/FigWorkAI/FigWorkAI
